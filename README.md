## 📋 Course Outline

| # | Topic | Key Concepts |
|---|-------|-------------|
| 1 | **Complex Numbers** | Algebraic & polar forms, modulus, argument, Euler's formula, De Moivre's Theorem |
| 2 | **Complex Functions** | Exponential, logarithmic & trigonometric functions, limits, continuity, differentiability |
| 3 | **Analytic Functions** | Cauchy–Riemann equations, harmonic functions, conformal mappings |
| 4 | **Complex Integration** | Cauchy's Integral Theorem, Cauchy's Integral Formula, contour integration |
| 5 | **Series & Singularities** | Taylor series, Laurent series, classification of singular points, poles |
| 6 | **Residue Calculus** | Residue Theorem, evaluation of real integrals using residues |
| 7 | **Laplace Transformations** | Definition & properties, inverse transforms, convolution, solving ODEs |
